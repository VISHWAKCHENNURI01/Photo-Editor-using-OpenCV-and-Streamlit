# 📸 Photo Editor using OpenCV and Streamlit

## Project Description

The **Photo Editor using OpenCV and Streamlit** is an interactive web application that enables users to perform various image editing operations through an easy-to-use interface. The application allows users to upload an image, apply multiple image enhancement techniques, preview the edited result in real time, and download the processed image.

Built using **Python**, **OpenCV**, and **Streamlit**, this project demonstrates practical applications of computer vision and digital image processing. It combines essential editing functionalities such as resizing, brightness and contrast adjustment, grayscale conversion, blur effects, sharpening, and portrait-style background blur with advanced filters including edge detection, sketch, cartoon, sepia, emboss, HDR enhancement, denoising, image rotation, and flipping.

The project provides a user-friendly interface where all image transformations can be controlled using sliders, checkboxes, and buttons, making it suitable for beginners learning OpenCV as well as users interested in basic photo editing.

## Steps to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/photo-editor-opencv-streamlit.git
```

### 2. Navigate to the Project Folder

```bash
cd photo-editor-opencv-streamlit
```

### 3. Install the Required Libraries

```bash
pip install -r requirements.txt
```

### 4. Run the Streamlit Application

```bash
streamlit run app.py
```

### 5. Open the Application

The application will automatically open in your default web browser.

If it does not open automatically, visit:

```
http://localhost:8501
```

### 6. Use the Application

- Upload an image.
- Adjust brightness and contrast.
- Resize the image.
- Apply filters and effects.
- Preview the edited image.
- Download the final edited image.
